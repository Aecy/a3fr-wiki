# abs

## Description

## Syntax

[Nombre](types/Nombre.md) = **abs** n

**Paramètres**
n: [Nombre](types/Nombre.md)

**Valeur retournée**
[Nombre](types/Nombre.md)

## Exemples

**Exemple 1**
```sqf
_nombre = abs -3;
```
