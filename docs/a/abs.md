# abs
abs

## Description
Valeur absolue d'un vrai nombre

## Syntax

**Syntax**<br>
[Nombre](types/Nombre.md) = **abs** n

**Paramètres**<br>
n: [Nombre](types/Nombre.md)

**Valeur retournée**<br>
[Nombre](types/Nombre.md)

## Exemples

**Exemple 1**
```sqf
_nombre = abs -3;
```
